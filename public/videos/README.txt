HERO VIDEO — drop your file(s) here
====================================

Put your hero video in this folder named exactly:

  hero.mp4        (required — H.264/AAC, the universal format)
  hero.webm       (optional — VP9/AV1, smaller; loaded first if present)

The site automatically fades the video in over the existing hero image.
If no file is here, the hero just shows the still image. Nothing breaks.

RECOMMENDED SPECS
-----------------
- Aspect ratio : 16:9, horizontal (landscape)
- Resolution   : 1920x1080 (1080p). 2560x1440 if you want extra crispness.
- Duration     : 6-12 seconds, designed to loop seamlessly (first and last
                 frame should match so the loop is invisible).
- Frame rate   : 24 or 30 fps
- File size     : keep under ~6 MB for fast load (compress it). 8 MB max.
- Audio         : none needed (the video is muted + autoplay). Strip the
                 audio track to save size.
- Motion        : slow and subtle. Autoplay only works when muted, so think
                 "moving photograph", not an ad.

WHAT TO FILM / GENERATE (to match the current hero)
---------------------------------------------------
A frosted-glass serum bottle with a champagne-gold dropper labeled
"RICHY RISK" on the RIGHT third of the frame, on cream travertine marble
with soft draped ivory silk. Warm directional light from the right, golden
glow, shallow depth of field. The LEFT half stays dark/empty negative space
so the white headline reads cleanly on top.

Good motions to capture (pick one, keep it gentle):
  - a single drop falling from the dropper / a slow serum drip
  - silk fabric breathing slightly in a breeze
  - a slow push-in (very slow dolly toward the bottle)
  - light shimmer slowly crossing the gold cap and glass
  - condensation / glaze slowly sliding down the bottle

Keep the framing and colors close to /public/images/hero-v2.png so the
fade from image to video is seamless.

AFTER ADDING THE FILE
---------------------
Just save it here and reload the page. No code changes needed.
